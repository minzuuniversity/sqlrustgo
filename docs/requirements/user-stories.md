# 用户故事
## US-001: 执行SELECT查询
作为 用户
我想要 执行SELECT查询语句
以便于 从数据库中检索数据
验收标准：
- 支持基本的SELECT语句
- 支持WHERE条件
- 支持ORDER BY排序

## US-002: 执行INSERT操作
作为 用户
我想要 执行INSERT操作语句
以便于 向数据库表中插入数据
验收标准：
- 支持基本的INSERT INTO ... VALUES语句
- 支持多列数据插入
- 插入失败返回明确错误信息

## US-003: 执行UPDATE操作
作为 用户
我想要 执行UPDATE更新语句
以便于 修改数据库中已存在的数据
验收标准：
- 支持带WHERE条件的精准更新
- 支持多列同时更新
- 无匹配数据时提示更新结果

## US-004: 执行DELETE操作
作为 用户
我想要 执行DELETE删除语句
以便于 删除数据库中指定数据
验收标准：
- 支持带WHERE条件的精准删除
- 禁止无WHERE条件的全表删除
- 删除成功返回受影响行数
